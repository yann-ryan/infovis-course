Tidyverse Exercises

This set of exercises will allow you to practise the Tidyverse commands and functions we have learned in the class so far. If you press ‘run code’ it will check against the given solution. You can also view the correct solution.

Of course, there are many ways to complete these problems. This grading software checks the answer rather than the desired output, which means it might not match up exactly with your solution, so bear that in mind!

Note that objects created do not carry over from one exercise to the next.

Titanic dataset exercises

Use tidyverse commands to carry out the following data analysis.

Use the filter() function to filter passengers by gender.

titanic |> filter(sex == "female") 
titanic |> filter(sex == "female")

Exercise 2

titanic |> filter(sex == "male") |> summarize(mean_age = mean(age, na.rm = TRUE))

  filter(sex == "male") |> 
  summarize(mean_age = mean(age, na.rm = TRUE))
